Download Fabric

Install Fabric API

Download the Fabric API and keep it ready to place in your mods folder.

3

Prepare Mods Folder

Navigate to %appdata%/.minecraft/mods (Windows) or the equivalent mods folder on Linux.

4

Add Veltragossa & API

Place both Veltragossa.jar and Fabric API into the mods folder for Minecraft 1.21.4.
